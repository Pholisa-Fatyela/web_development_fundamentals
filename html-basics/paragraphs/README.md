# Paragraphs

## The &lt;p&gt; Element

To create a paragraph, simply type in the **&lt;p&gt;** element with its opening and closing tags:

```markup
<html>
 <head>
 <title>first page</title>
 </head>
 <body>
 <p>This is a paragraph. </p>
 <p>This is another paragraph. </p>
 </body>
</html>
```
